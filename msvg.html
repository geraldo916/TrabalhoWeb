O que é Mathml

Mathmatical Markup Language é uma linguagem de marcação matemática, uma aplicação de XML para descrever noções matmáticas e capturar sua estrutura e conteúdo. Parte do HTML5 e ISO desde 2015, o Mathml visa intefrar de fórmulas matemáticas em páginas Word Wide Web e outros documentos.

Equação do Segundo Grau

A equação do segundo grau recebe esse nome porque é uma equação polinomial cujo termo de maior grau está elevado ao quadrado. Também chamada de equação quadrática, é representada por:

ax2 + bx + c = 0

Numa equação do 2º grau, o x é a incógnita e representa um valor desconhecido. Já as letras a, b e c são chamadas de coeficientes da equação.

O que é SVG

SVG é a abreviatura de Scalable Vector Graphics que pode ser traduzido do inglês como gráficos vetorias escalonáveis. Trata-se de uma linguagem XML para descrever de forma vetorial desenhos e gráficos bidimensionais, quer de forma estática, quer dinâmica ou animada.
