HTML

O HTML é um conjunto de tags responsáveis pela marcação do conteúdo de uma página no navegador. No código que vimos antes, as tags são os elementos a mais que escrevemos usando a sintaxe <'nomedatag'> . Diversas tags são disponibilizadas pela linguagem HTML e cada uma possui uma funcionalidade específica.

CSS 3

O CSS formata a informação entregue pelo HTML. Essa informação pode ser qualquer coisa: imagem, texto, vídeo, áudio ou qualquer outro elemento criado. Grave isso: CSS formata a informação. Essa formatação na maioria das vezes é visual, mas não necessariamente.

JavaScript

Desenvolvida pela NETSCAPE, a linguagem JavaScript foi criada para trabalhar com aplicações interativas nas páginas HTML. Esta linguagem teve sua primeira versão desenvolvida para o browser Netscape Navigator 2.0

Além de HTML CSS e Javascript existem outras diversas ferramentas para a criação de webpages como frameworks, e outras linguagens.

Apesar de ser menos popular que os demais frameworks, o Vue.js reúne mais características de segurança e agilidade. Por conta disso, ele é muito utilizado pelo time da Decode. Na linguagem Javascript, um framework interessante é o React, do Facebook. Ele possibilita criações simples para cada estado de aplicação. Além disso, é ágil, intuitivo e tem sintaxe de alta performance para conteúdo. É o framework mais amplamente utilizado no mercado

Bootstrap e um framework web com código-fonte aberto para desenvolvimento de componentes de interfaces e front-end para sites e aplicações web usando HTML, CSS e Javascript, baseado em modelos de design para a tipografia, melhorando a experiencia do usuario em um site amigável e responsivo
